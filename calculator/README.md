# Python Calculator

A beginner Python calculator built while learning Python fundamentals.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Exponentiation (**)
- Square Root (sqrt)

## Error Handling

- Division by zero protection
- Modulus by zero protection
- Negative square root validation

## Technologies Used

- Python 3
